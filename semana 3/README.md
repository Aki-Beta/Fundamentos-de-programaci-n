# Sistema de Gestión de Inventario 

> Este proyecto es una aplicación de consola desarrollada en Python para gestionar el inventario de una tienda tecnológica. Permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar), calcular estadísticas de ventas y asegurar la persistencia de los datos mediante archivos CSV.

## Características Principales

* Gestión de Productos: Registro de nombre, precio y cantidad.

* Persistencia de Datos: Guardado y carga de información desde archivos .csv.

* Lógica de Fusión: Al cargar archivos, el sistema permite sobrescribir el inventario actual o fusionarlo (sumando stock de productos existentes).

* Estadísticas en Tiempo Real: Cálculo de unidades totales, valor del inventario y detección del producto más costoso.

* Modularización: Código organizado en capas (app.py, servicios.py, archivos.py) para facilitar el mantenimiento.

* Manejo de Errores: Validaciones con try/except para prevenir cierres inesperados por datos inválidos.

##  Tecnologías Utilizadas

>*  Lenguaje: Python 3.10+

>* Formato de Datos: CSV (Valores separados por comas).

>* Diagramación: Draw.io (Diagrama de flujo incluido en la entrega).
